# SF-WhatDreamsCost-ComfyUI

基于 [WhatDreamsCost/WhatDreamsCost-ComfyUI](https://github.com/WhatDreamsCost/WhatDreamsCost-ComfyUI) 和 [CS-WhatDreamsCost-ComfyUI](https://github.com/yg496/CS-WhatDreamsCost-ComfyUI) 二次开发。

SF 版本核心增强：**自动识别万象分镜脚本视频提示词中的宫格位置标注和时长标注**，实现音画时长同步、视频和谐连贯。

## SF 版本核心增强

### 1. 宫格位置标注自动识别

SF 版本新增 `_segments_from_grid_text` 解析函数，能自动识别视频提示词中的宫格位置标注（如"左上，""中上，""右上传"等），将单段连续文字自动拆分为对应宫格位置的提示词。

**支持的宫格位置标注：**
- 4宫格：左上、右上、左下、右下
- 6宫格：左上、中上、右上、左下、中下、右下
- 9宫格：左上、中上、右上、左中、中中、右中、左下、中下、右下

### 2. 时长标注自动解析

SF 版本能自动解析视频提示词中每格的时长标注（如"4秒，"），将时长信息传递给导演台时间线，实现非平均时长分配。

**时长分配原则：**
- 有台词的格时长长一些（保证台词能说完）
- 近景和特写时长长一些（展示表情细节）
- 中景时长短一些（仅交代位置）
- 所有格时长之和等于锁定总时长（4宫格=12秒，6宫格=18秒，9宫格=27秒）

### 3. 万象分镜脚本兼容

SF 版本完全兼容万象软件的视频提示词格式：
- 视频提示词是一段连续文字（万象软件可识别）
- 宫格位置标注和时长标注内联在文字中
- 万象软件读取整段文字作为提示词，不影响识别
- ComfyUI插件自动解析其中的宫格位置和时长信息

## 视频提示词格式示例

```
视频提示词:电影感真人摄影，现代都市剧照，便服日常装，都市室内与街市场景，自然光与暖黄室内光，浅景深，35mm镜头质感，无文字无水印无边框。第一个镜头（对应3x3九宫格从左上至右下区域）：左上，中景，3秒，李强坐在床边翻看彩票。中上，近景，4秒，李强低头看手机屏幕，眉头皱起。右上，近景，3秒，李强手指指向彩票数字。[李强："全对上了！"]
```

**格式说明：**
1. 开头是风格提示词
2. 然后是镜头序号标注："第一个镜头（对应[宫格布局]从左上至右下区域）："
3. 每格描述格式："[宫格位置]，[景别]，[时长]秒，[描述内容]"
4. 各格之间用句号"."分隔
5. 整段是一段连续文字，不能有空行或换行

## 节点说明

核心节点：

| 节点 ID | 显示名称 | 作用 |
| --- | --- | --- |
| `SF-LTXGridDirector` | `SF-LTX 宫格导演台` | 从四宫格、六宫格或九宫格图像自动创建可编辑的 LTX 导演台时间线，支持自动识别宫格位置标注和时长标注。 |
| `SF-LTXDirectorGuide` | `SF-LTX 导演台引导` | 将导演台输出的图像引导数据接入 LTX 条件和潜空间链路。 |

`SF-LTX 宫格导演台` 是原六宫格导演台的升级版，支持 `2x2 四宫格`、`3x2 六宫格`、`3x3 九宫格`，并增加：
- 分镜比例裁剪、白边裁剪强度
- 图像源变化自动刷新和旧工作流参数自修复
- **SF新增：宫格位置标注自动识别（左上、中上、右上...）**
- **SF新增：时长标注自动解析（如"4秒，"）**
- **SF新增：万象分镜脚本视频提示词兼容**

## 基本流程

1. 准备一张四宫格、六宫格或九宫格参考图。
2. 将参考图连接到 `宫格图像`。
3. 将万象分镜脚本生成的视频提示词连接到 `GPT 分镜文本`。
4. 选择 `宫格模式`，例如 `3x3 九宫格`。
5. 选择 `分镜比例`，例如 `16:9 横屏` 或 `9:16 竖屏`。
6. 根据宫格边框情况调整 `白边裁剪强度`。
7. 点击 `自动填充宫格`，节点会自动识别视频提示词中的宫格位置标注和时长标注，生成对应数量的可编辑分镜块，并按时长分配每段帧数。
8. 在导演台前端检查并手动修改每段分镜文本、时长或顺序。
9. 将 `引导数据` 接到 `SF-LTX 导演台引导`，继续连接 LTX 采样和解码节点。

## 宫格模式

拆分顺序始终为从左到右、从上到下。

| 模式 | 分镜数量 | 顺序 |
| --- | ---: | --- |
| `2x2 四宫格` | 4 | 左上、右上、左下、右下 |
| `3x2 六宫格` | 6 | 上排 1-3，下排 4-6 |
| `3x3 九宫格` | 9 | 第一排 1-3，第二排 4-6，第三排 7-9 |

## 分镜比例

`分镜比例` 控制每个宫格单元拆出来之后的最终画面比例。

| 选项 | 说明 |
| --- | --- |
| `自动 / 保持单格比例` | 保持原始宫格单元比例，最稳妥。 |
| `16:9 横屏` | 将每个分镜居中裁切为横屏比例。 |
| `9:16 竖屏` | 将每个分镜居中裁切为竖屏比例。 |
| `1:1 方图` | 将每个分镜居中裁切为正方形。 |

## 文本输入建议

SF 版本支持三种文本格式：

### 1. 万象分镜脚本格式（推荐）

一段连续文字，包含宫格位置标注和时长标注：

```text
电影感真人摄影，现代都市剧照。第一个镜头（对应3x3九宫格从左上至右下区域）：左上，中景，3秒，李强坐在床边翻看彩票。中上，近景，4秒，李强低头看手机屏幕。右上，近景，3秒，李强手指指向彩票数字。[李强："全对上了！"]
```

### 2. JSON 格式

```json
[
  {"prompt": "中景，李强坐在床边", "seconds": 3},
  {"prompt": "近景，李强看手机屏幕", "seconds": 4}
]
```

### 3. 编号文本格式

```text
分镜1：中景，李强坐在床边
分镜2：近景，李强看手机屏幕
```

## 安装

将仓库放入 ComfyUI 的 `custom_nodes` 目录，然后重启 ComfyUI：

```text
ComfyUI/custom_nodes/SF-WhatDreamsCost-ComfyUI
```

## 与小说转万象分镜脚本技能配合使用

1. 使用 `novel-to-wanxiang-storyboard` 技能（V2.3+）生成分镜脚本
2. 将生成的视频提示词复制到 ComfyUI 的 `SF-LTX 宫格导演台` 节点的 `GPT 分镜文本` 输入框
3. 插件自动识别宫格位置标注和时长标注
4. 自动按时长分配每段帧数，确保音画同步

## 使用注意事项

- `每段帧数` 输入框**留空**，插件会自动用视频提示词中的时长标注分配帧数；若手动填写则覆盖自动分配。
- `总帧数` 应与锁定总时长匹配：4宫格12秒=288帧(24fps)、6宫格18秒=432帧、9宫格27秒=648帧。
- 视频提示词必须是**一段连续文字**，宫格位置标注（左上/中上/...）和时长标注（N秒，）内联其中，不能换行分段。

## 更新日志

### v2.1.0（2026-06-20）

**修复**：前端残留 `timeline_data` 导致宫格解析被跳过的严重问题。

- **问题现象**：实际使用时所有提示词挤在第一个镜头、时长被平均分配。
- **根因**：ComfyUI 前端时间线编辑器默认会塞入 segments，使后端 `if not timeline["segments"]` 条件为假，`_build_default_timeline`（宫格解析核心）永远不被触发。
- **修复**：`SF-LTX 宫格导演台` 检测到视频提示词包含宫格位置标注时，**强制重建时间线**，忽略前端残留的 `timeline_data`。
- **优化**：`SF-LTX 自动导演台` 检测到宫格格式时用解析出的分镜数量覆盖 `segment_count`，避免手动设置不匹配。
- **修复**：`SF-LTX 自动导演台` 显示名从残留的 `CS LTX Auto Director` 改为 `SF-LTX 自动导演台`。

### v2.0.0

- 从 CS- 前缀改为 SF- 前缀，重命名为 SF-WhatDreamsCost-ComfyUI。
- 新增 `_segments_from_grid_text` 解析函数，识别宫格位置标注和时长标注。
- `MAX_AUTO_SEGMENTS` 从 6 扩展到 9，支持九宫格。
- 新增测试套件 `test_grid_parser.py`（6 个用例）。
